# RayTracer
Creating ray tracer based on Peter Shirley's series "Ray Tracing in One Weekend". At this moment it's render images in .ppm format.

Development goals:
1. Metal Materials
2. Dielectric materials
3. Positionable Camera
4. Defocus Blur
5. Motion Blur
6. Bounding Volume Hierarchies
7. Solid Textures
8. Perlin Noise
9. Image Texture Mapping
10. Rectangles and Lights
11. Instances
12. Volumes

Potential development goals:
1. Implementing ray tracer with OpenGL
